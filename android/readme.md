
## 操作步骤

1. 安装virtual-xposed apk地址：https://github.com/paopaosecret/resources/blob/master/apk/app-aosp-debug.apk

3. 在virtual-xposed 安装该hook apk 并启动
4. 在Virtual-xposed 中运行需要hook的应用，进行测试。
